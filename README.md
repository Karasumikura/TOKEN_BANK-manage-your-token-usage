## 核心特性

| 特性 | 说明 |
|:---|:---|
| **100% 本地** | 所有数据本地读取处理，不经过任何外部服务器 |
| **开箱即用** | 单个 `app.py` 即完整应用，也可打包为单个 `.exe`，拷贝即用 |
| **多平台聚合** | 同时支持 Claude Code / Codex / Cline 会话数据 |
| **精细计费** | 自定义模型单价，Input / Output / Cache Read 三维度 |
| **时间段对比** | 一键预设（今天 vs 昨天、本周 vs 上周等），差异清晰呈现 |
| **智能预算** | 每日/每月费用上限，支持按应用和按模型独立设置 |
| **数据导出** | 一键 CSV 导出，自动生成使用摘要报告 |
| **迷你窗口** | 紧凑置顶视图，实时监控核心指标 |
| **原生体验** | pywebview 桌面窗口，系统托盘常驻，深色/浅色主题，中英双语 |

## 功能详情

### 总览面板

- **6 项核心指标** — 总记录数、输入 Token、输出 Token、缓存读取、预估费用、缓存命中率
- **每日趋势图** — 堆叠柱状图展示 Input / Output / Cache Read 分布
- **费用分布** — 环形图按模型拆分费用占比
- **按应用/模型统计** — 维度汇总表，彩色标签区分
- **效率指标** — 每条消息平均 Token / 费用
- **费用预测** — 基于近 7 天趋势推算本月预估费用

### 使用报告

- **日期范围** — 自定义起止日期，1D / 7D / 14D / 30D / 全部快捷预设
- **趋势图表** — Token 用量折线图 + 日费用折线图
- **逐小时钻取** — 选中单日自动切换 24 小时粒度视图
- **使用热力图** — 短期按日期 × 小时，长期按星期 × 小时
- **模型排行** — 各模型各时段 Token 用量横向对比，从多到少排列
- **模型趋势** — 各模型使用量占比随时间变化

### 项目 / 会话

- 按项目维度汇总用量，自动解码 Claude Code 项目路径
- 会话 Top 100 排行，支持搜索、筛选、展开查看完整消息流

### 对比分析

- **一键预设** — 今天 vs 昨天、本周 vs 上周、本月 vs 上月
- **差异指标** — 增减幅度 + 方向箭头
- **分组柱状图** — 双时段并排对比

### 预算管理

- **双模式** — Token 预算 / 费用预算独立切换
- **全局限额** — 每日 + 每月，环形进度可视化
- **按应用/模型限额** — 独立设置，使用率 + 占比双指标
- **超限预警** — 80% / 100% 阈值自动提醒

### 设置

- 中英双语切换 · 深色/浅色主题
- 模型计费编辑器（保存到 `~/.tokenbank/pricing.json`）
- 自动刷新（15s / 30s / 60s / 120s）

## 自定义模型计费

支持为每个模型独立设置 Input / Output / Cache Read 三维单价（$/百万 Token）。内置主流模型默认价格，可在设置中随时修改、添加新模型或恢复默认。未匹配模型使用默认计费。

## 快速开始

**方式一：直接使用 EXE（无需 Python 环境）**

`dist/TOKENBANK.exe` 为单文件免安装版本，可直接拷贝到任意 Windows 电脑运行。

**方式二：从源码运行**

```bash
pip install -r requirements.txt
python app.py
```

也可双击 `TOKENBANK.bat` 启动（自动优先使用 pythonw 隐藏控制台窗口）。

**方式三：自行打包 EXE**

```bash
python build.py    # 产出 dist/TOKENBANK.exe
```
