# TOKENBANK

**本地 AI 编程助手 Token 用量统计与费用管理面板**

## 亮点

- **100% 本地，零上传** — 所有数据在本地读取和处理，不经过任何外部服务器，隐私安全有保障
- **开箱即用，单文件运行** — 一个 `app.py` 即完整应用，也可打包为单个 `.exe`，免安装拷贝即用
- **多平台聚合** — 同时支持 Claude Code 和 Codex 的会话数据，在同一面板中统一查看
- **精细费用追踪** — 支持自定义模型计费单价，精确到 Input / Output / Cache Read 三维度，费用一目了然
- **时间段对比** — 任意两个时间段横向对比，支持一键预设（今天 vs 昨天、本周 vs 上周等），差异指标清晰呈现
- **原生桌面体验** — 基于 pywebview 原生窗口渲染，支持系统托盘常驻、深色/浅色主题、中英双语切换
- **智能预算告警** — 设置每日/每月费用上限，接近或超出时自动提醒，费用可控
- **数据导出与报告** — 一键导出 CSV，自动生成使用摘要报告，方便分享与分析
- **迷你桌面窗口** — 紧凑置顶视图常驻桌面角落，实时监控核心指标

## 功能详解

### 总览

全局概览面板，一次掌握全部用量数据：

- **6 项核心指标卡片** — 总记录数、输入 Token、输出 Token、缓存读取、预估费用、缓存命中率，每张卡片带彩色顶边标识，关键数据一眼可见
- **每日趋势图** — 堆叠柱状图展示每日 Input / Output / Cache Read / Cache Create 的分布走势
- **费用分布图** — 环形图按模型拆分费用占比（多模型时自动显示）
- **按应用统计表** — 按 Claude Code、Codex 等应用维度汇总，带彩色标签区分
- **按模型统计表** — 按模型维度汇总用量与费用
- **最近消息** — 最近 10 条消息的快速预览
- **效率指标** — 每条消息平均 Token、每条消息平均费用、每千 Token 平均费用
- **费用预测** — 基于近 7 天趋势推算本月预估总费用

### 使用报告

按时间维度深入分析用量变化：

- **日期范围选择** — 支持自定义起止日期，附带 1D / 7D / 14D / 30D / 全部 快捷预设
- **Token 用量折线图** — 展示 Input、Output、Cache Read 随时间的变化趋势
- **日费用折线图** — 展示每日费用走势
- **明细表格** — 按日聚合的详细数据表
- **逐小时钻取** — 选择单日时自动切换为 24 小时粒度的逐小时视图，精准定位高消耗时段
- **使用热力图** — 按星期 x 小时生成热力图，直观展示高峰使用时段
- **模型趋势图** — 折线图展示各模型使用量占比随时间的变化走势

### 项目

按项目维度统计用量：

- 自动解码 Claude Code 的项目路径编码（如 `C--Users-foo-project` → `C:/Users/foo/project`）
- 展示每个项目的会话数、消息数、输入/输出/缓存读取 Token、总 Token、预估费用

### 会话

按会话维度追踪单次对话的消耗：

- **Top 100 排行** — 按总 Token 用量降序排列，快速定位最"烧 Token"的会话
- 展示应用标签、使用模型、日期、消息数、输入/输出 Token、费用
- **会话摘要** — 自动提取会话的主题摘要信息
- **会话详情** — 点击任意会话行展开查看完整消息流和逐条 Token 消耗
- **搜索过滤** — 支持按关键词、应用、模型筛选会话

### 对比

任意两个时间段横向对比，洞察用量变化趋势：

- **一键预设** — 今天 vs 昨天、本周 vs 上周、本月 vs 上月，点击即出结果
- **自定义区间** — 也可手动选择任意两个日期范围
- **差异指标卡片** — 总输入、非缓存输入、输出、缓存读取、费用、记录数，每项显示增减幅度与方向箭头
- **分应用对比表** — 两个时段各自的按应用明细
- **分组柱状图** — 5 项核心指标的双时段并排对比图

### 设置

- **语言切换** — 中文 / English，全界面即时切换，设置自动保存
- **主题切换** — 深色 / 浅色主题，设置自动保存
- **数字格式** — 中式单位（万/亿）或西式单位（K/M/B），跟随语言自动适配，也可手动切换
- **模型计费编辑器** — 自由增删模型、编辑各模型的 Input / Output / Cache Read 单价（$/百万Token），保存到本地 `~/.tokenbank/pricing.json`，支持一键恢复默认
- **自动刷新** — 定时自动从磁盘重新加载数据，默认 30 秒，可在 15s/30s/60s/120s 间切换，支持开关
- **预算告警** — 设置每日/每月费用上限，接近（80%）或超出时在顶部显示警告
- **手动刷新** — 从磁盘重新读取全部会话数据并刷新面板

### 数据导出

- **导出 CSV** — 概览、使用报告、项目、会话、对比各页面均支持一键导出当前数据为 CSV 文件
- **使用报告** — 自动生成近 7 天/30 天/全部的文字摘要报告，包含总量、日均、热门模型、热门项目，支持一键复制

### 迷你模式

- **桌面迷你窗口** — 点击顶部 Mini 按钮切换为紧凑迷你视图，窗口缩小并置顶，仅显示核心指标（输入/输出/费用），适合常驻桌面角落监控

## 截图

> 轻量级桌面应用，基于 pywebview 原生窗口渲染，无需浏览器。支持系统托盘常驻，关闭窗口时可选择最小化到托盘或退出。

## 快速开始

```bash
# 安装依赖
pip install -r requirements.txt

# 运行
python app.py
```

也可直接双击 `TOKENBANK.bat` 启动（自动优先使用 pythonw 隐藏控制台窗口）。

## 打包为 EXE

```bash
# 方式一：Python 脚本
python build.py

# 方式二：批处理
build.bat
```

打包完成后在 `dist/` 目录生成 `TOKENBANK.exe`，单文件免安装，可直接拷贝到任意 Windows 电脑运行（无需 Python 环境）。

## 数据源

自动读取以下路径的会话数据：

| 应用 | 路径 |
|------|------|
| Claude Code | `~/.claude/projects/*/sessions/*.jsonl` |
| Codex | `~/.codex/sessions/*.jsonl` |

数据仅在本地读取和处理，不会上传到任何服务器。

## 内置模型计费

| 模型 | Input ($/百万Token) | Output ($/百万Token) | Cache Read ($/百万Token) |
|------|-----|------|------|
| claude-opus-4-7 | 5.00 | 25.00 | 0.50 |
| claude-opus-4-6 | 5.00 | 25.00 | 0.50 |
| claude-opus-4-5 | 5.00 | 25.00 | 0.50 |
| claude-sonnet-4-6 | 3.00 | 15.00 | 0.30 |
| claude-sonnet-4-5 | 3.00 | 15.00 | 0.30 |
| claude-haiku-4-5 | 1.00 | 5.00 | 0.10 |
| mimo-v2-pro | 1.00 | 3.00 | 0.00 |
| mimo-v2.5-pro | 0.00 | 0.00 | 0.00 |
| gpt-5.4 | 2.50 | 15.00 | 0.25 |

未匹配的模型将使用默认计费：Input $3.00 / Output $15.00 / Cache Read $0.30。所有单价均可在设置中自定义修改。

## 技术栈

- **Python** — 数据读取与聚合
- **pywebview** — 原生桌面窗口（Windows 上使用 WebView2）
- **Chart.js** — 图表渲染（柱状图、折线图、环形图）
- **pystray** — 系统托盘图标与菜单
- **Pillow** — 程序化生成托盘图标
- **PyInstaller** — 单文件 EXE 打包

## 系统要求

- Windows 10+（需 WebView2 运行时，Win10 21H2+ 已内置）
- Python 3.9+
